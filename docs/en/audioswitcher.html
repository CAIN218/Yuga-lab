AudioSwitcher

A lightweight, standalone audio device switching utility for Windows

Overview

AudioSwitcher is a resident Windows utility built for practical, everyday use: instantly reroute audio playback devices — built-in display speakers, headsets, external DACs, and more — with a single global shortcut key. It removes the hassle of digging through Control Panel or the taskbar, giving you seamless output switching mid-game or anywhere in your workspace.

Key Features

■ A fade notification synced to the taskbar

When a switch is triggered, a smooth fade notification overlays precisely on the taskbar clock area in the bottom-right corner. The design goal is extreme minimalism — never getting in the way of your work or on-screen content.

■ No more conflicts between identically-named devices

Windows frequently misidentifies audio devices that share the same product name. AudioSwitcher solves this by reading each device's persistent unique identifier directly from the Windows Core Audio API (pycaw), keeping routing 100% accurate without relying on any external tool.

■ Smart auto-registration for startup

Built-in logic for registering itself in the Windows startup folder. It automatically detects whether it's running as a raw script or a compiled single-file binary (exe), and gets a stable background process running immediately either way.

Package Structure

  • AudioSwitcher.exe — the whole application, self-contained down to device control. No external tools needed.
